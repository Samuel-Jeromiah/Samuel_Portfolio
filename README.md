## üöÄ Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/LuminarySam30/Samuel_Portfolio.git
    cd Samuel_Portfolio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open your browser**:
    Visit [http://localhost:3000](http://localhost:3000) to see the application live.
